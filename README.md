# NTHU Vehicle Permit Sticker Studio

Studio for generating NTHU vehicle permit stickers.

## Preview

![NTHU Vehicle Permit Sticker Studio preview](docs/images/preview.png)

## Features

- Generate two permit types:
  - Bicycle (`7 x 3 cm`)
  - Motorcycle (`6.2 cm` diameter)
- Customize:
  - Theme color
  - Surface tone (dark or light)
  - Academic year
  - Serial text (preview)
  - Custom logo upload (`SVG`, `PNG`, `JPG`)
  - Logo scale and tint settings per vehicle type
- Export print-ready assets in `SVG`, `PNG`, and `JPG`
